Episoder

Download this project as a .zip file Download this project as a tar.gz file

episoder is a tool to tell you about new episodes of your favourite TV shows. It does so by parsing online TV show episode guides.

Currently episoder can get data from epguides and TheTVDB and tells you whenever a new episode of a show listed on one of those sites is aired.

There is also a web-based version of episoder, called webisoder.

Download

You will probably want to download the latest release. The source code for episoder is available under the terms of the GNU General Public License (Version 3) and can be downloaded through github.

Packages

There are also packages available for Debian and Ubuntu.

Release notification

To stay informed about new releases, please subscribe to episoder on the freecode project page.
